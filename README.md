# 🌐 SHAMS Portfolio

This project is a **responsive personal portfolio website** built with **HTML**, **CSS**, and **JavaScript**.
It was created to present my background, skills, projects, education, CV, and contact information in a structured and interactive way.

The website includes several pages such as:

* Home
* About Me
* Experience
* Skills
* Projects
* Education
* Interests
* CV
* Contact Me

##  CV Page

The CV page presents my CV in a simple and structured way.

### Multilingual Support

The page supports:

* Swedish
* English

### Language Functionality

* Both languages are included in the HTML
* JavaScript shows only the selected language
* English is used as the default language
* The selected language is saved in `localStorage`
* The preference remains after page reload

This improves accessibility and makes the portfolio easier to use for different audiences.
